🎵 Simple Music App (API Based)
A lightweight and user-friendly Music Streaming App built using modern development practices.
This application fetches music data from a public API and allows users to browse and play songs seamlessly.

📱 Features
1. 🎶 Fetch songs dynamically using API
2. 🔍 Browse music list with album art
3. ▶️ Play / Pause functionality
4. ⏭ Next / Previous track controls
5. 📡 Real-time data loading
6. 📱 Clean and responsive UI

🛠️ Tech Stack
Language: Java / Kotlin (Update based on what you used)
IDE: Android Studio
API Integration: REST API (e.g., Retrofit / Volley)
JSON Parsing
MediaPlayer for audio streaming
RecyclerView for listing songs
